# UI/UX Specifications

## Visual Identity
- **Theme:** Strict Dark Mode (OLED Black #000000).
- **Primary Accent:** Cyber-Lime (#CCFF00) for "Start Workout" and active elements.
- **Secondary Accent:** Slate Grey (#2C2C2E) for cards.
- **Typography:** System Sans-Serif (Inter/Roboto), heavy weights for numbers (weights/reps).

## Navigation (Bottom Bar)
1. **Dashboard:** Summary of current week vs "The Committee's" goals.
2. **Log:** The active workout logger (Main screen).
3. **History:** Calendar view of past sessions.
4. **War Room:** The multi-persona AI chat interface.
