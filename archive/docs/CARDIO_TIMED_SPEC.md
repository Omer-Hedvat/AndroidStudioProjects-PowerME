# Cardio & Timed Exercises

## 1. Exercise Types
- **Strength**: Weight x Reps (Existing).
- **Cardio**: Distance x Time (Pace is calculated: $Pace = \frac{Time}{Distance}$).
- **Timed**: Time x (Optional) Weight (e.g., Weighted Plank).

## 2. Integration
- Sync Cardio sessions to Health Connect as `WorkoutSession`.
- Coach Carter analyzes Cardio sessions to adjust the 'Recovery Score' in the War Room.
